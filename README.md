# Merton Jump Diffusion

Extending Black–Scholes/GBM model by adding Poisson jumps to capture sudden market moves.

This project is part of the Erdős Institute Quant Finance Boot Camp. The contents are based on the course given by Thomas Polstra in the Fall of 2025.
